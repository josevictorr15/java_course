
• É uma estrutura de controle que permite definir que um certo bloco de comandos somente será executado dependendo de uma condição.

• Sintaxe :

```
if (<condição>) {
	<comando 1>
	<comando 2>
}
```

• Sintaxe composta :

```
if (<condição>) {
	<comando 1>
	<comando 2>
} else {
	<comando 3>
	<comando 4>
}
```
